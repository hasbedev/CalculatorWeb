# CalculatorWeb
Calculator X HasbeDev adalah aplikasi kalkulator berbasis web dengan desain minimalis dan fokus pada kenyamanan pengguna. 
Dilengkapi dengan dukungan keyboard, riwayat perhitungan, mode gelap/terang, serta antarmuka sederhana tanpa elemen berlebihan. 
Dibangun sebagai PWA sehingga dapat di-install dan digunakan layaknya aplikasi native.
